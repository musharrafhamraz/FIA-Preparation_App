The quiz app needs a database of MCQs to work. Run the scraper first:

Step 1. Install Python libraries:
pip install requests beautifulsoup4

Step 2. Run the scraper (downloads everything):
python scraper.py
For a quick test: python scraper.py --pages 10

Step 3. Place questions.json in the same folder as quiz.html,
then open quiz.html in your browser.

⚠️ You must open quiz.html via a local server or Firefox (Chrome blocks local file fetch).
Quick server: python -m http.server 8000 then visit localhost:8000/quiz.html
